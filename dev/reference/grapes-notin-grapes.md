# Negated Value Matching

Returns a `logical` vector indicating if there is *no* match of the left
operand in the right operand.

## Usage

``` r
x %notin% table
```

## Arguments

- x:

  The values to be matched

  Default value

  :   none

- table:

  The values to be matched against

  Default value

  :   none

## Value

A `logical` vector
