# check that argument contains valid variable(s) created with `exprs()` or Source Variables from a List of Expressions

check that argument contains valid variable(s) created with
[`exprs()`](https://rlang.r-lib.org/reference/defusing-advanced.html) or
Source Variables from a List of Expressions

## Usage

``` r
contains_vars(arg)
```

## Arguments

- arg:

  A function argument to be checked

  Default value

  :   none

## Value

A `TRUE` if variables were valid variable
