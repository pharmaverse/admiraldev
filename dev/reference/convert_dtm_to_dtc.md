# Helper Function to Convert Date (or Date-time) Objects to Characters of dtc Format (-DTC type of variable)

Helper Function to Convert Date (or Date-time) Objects to Characters of
dtc Format (-DTC type of variable)

## Usage

``` r
convert_dtm_to_dtc(dtm)
```

## Arguments

- dtm:

  date or date-time

  Default value

  :   none

## Value

`character` vector
