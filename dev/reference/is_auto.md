# Checks if the argument equals the auto keyword

Checks if the argument equals the auto keyword

## Usage

``` r
is_auto(arg)
```

## Arguments

- arg:

  argument to check

  Default value

  :   none

## Value

`TRUE` if the argument equals the auto keyword, i.e., it is an
expression of a symbol named auto.
