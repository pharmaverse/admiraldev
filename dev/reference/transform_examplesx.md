# Transform `@caption`, `@info` and `@code` tags into `@examplex` tags

Transform `@caption`, `@info` and `@code` tags into `@examplex` tags

## Usage

``` r
transform_examplesx(block)
```

## Arguments

- block:

  A block of roxygen tags of one function

  Default value

  :   none

## Value

The block with the transformed tags
