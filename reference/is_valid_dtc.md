# Is this string a valid DTC

Is this string a valid DTC

## Usage

``` r
is_valid_dtc(arg)
```

## Arguments

- arg:

  A `character` vector

  Default value

  :   none

## Value

`TRUE` if the argument is a valid `--DTC` string, `FALSE` otherwise
