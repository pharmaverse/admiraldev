# Warn if incomplete dtc

Warn if incomplete dtc

## Usage

``` r
warn_if_incomplete_dtc(dtc, n)
```

## Arguments

- dtc:

  A `character` vector of date-times in ISO 8601 format

  Default value

  :   none

- n:

  A non-negative integer

  Default value

  :   none

## Value

A warning if `dtc` contains any partial dates
