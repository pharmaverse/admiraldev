# Wrap a String in Double Quotes

Wrap a string in double quotes, e.g., for displaying character values in
messages.

## Usage

``` r
dquote(x)
```

## Arguments

- x:

  A character vector

  Default value

  :   none

## Value

If the input is `NULL`, the text `"NULL"` is returned. Otherwise, the
input in double quotes is returned.
