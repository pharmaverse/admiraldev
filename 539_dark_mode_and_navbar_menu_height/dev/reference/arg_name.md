# Extract Argument Name from an Expression

**\[deprecated\]**

This function is *deprecated*, please use
[`rlang::caller_arg()`](https://rlang.r-lib.org/reference/caller_arg.html)
instead.

## Usage

``` r
arg_name(expr)
```

## Arguments

- expr:

  An expression created inside a function using
  [`substitute()`](https://rdrr.io/r/base/substitute.html)

  Default value

  :   none

## Value

`character` vector
